## Purpose

This document tracks recurring symbols and motifs used across story, art, music, dungeons, dialogue, and UI.

Motifs should support emotional memory and cohesion without becoming heavy-handed.

## Motif Template

| Field | Purpose |
| --- | --- |
| Motif | Symbol, image, sound, phrase, object, action, or pattern. |
| Meaning | Emotional or thematic meaning. |
| First Appearance | Where it begins. |
| Variations | How it changes across the game. |
| Story Use | Plot or emotional function. |
| Character Use | Which characters connect to it. |
| Visual / Music Use | Art or audio treatment. |
| Canon Status | Draft, Review, Approved, Deprecated. |

## Potential Motif Areas

- Islands and crossings.
- Lanterns, stars, storms, tides, and paths.
- Old ruins and living homes.
- Offerings, inherited objects, and repaired things.
- Songs, names, and remembered phrases.
