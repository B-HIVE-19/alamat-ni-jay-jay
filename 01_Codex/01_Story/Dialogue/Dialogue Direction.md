## Purpose

This document defines the dialogue direction for *Alamat ni Jay Jay*.

Dialogue should reveal personality, relationship, history, humor, fear, growth, and conflict. It should not exist only to deliver exposition.

## Core Standard

Every major character must have a unique speaking style.

Each major character should become recognizable from dialogue alone. If the speaker's name is hidden, the player should still be able to infer who is speaking through rhythm, word choice, emotional habits, humor, silence, interruptions, and what the character notices.

## Dialogue Rules

1. Dialogue should reveal personality before information.
2. Avoid exposition whenever possible.
3. Let characters interrupt each other.
4. Let characters disagree without turning every disagreement into drama.
5. Let characters laugh, tease, argue, comfort, avoid, deflect, and grow.
6. Humor should emerge naturally from personality rather than jokes.
7. Characters should speak differently depending on trust, fear, fatigue, anger, grief, pride, and affection.
8. Dialogue should change as relationships evolve.
9. Avoid catchphrases as a shortcut for voice.
10. Silence, hesitation, and what a character refuses to say can be as revealing as spoken lines.

## Exposition Standard

When information must be communicated, prefer:

- Conflict over explanation.
- Discovery over lecture.
- Misunderstanding over summary.
- A character's biased perspective over neutral facts.
- Environmental storytelling before spoken explanation.
- A question that reveals motive before an answer that explains lore.

## Conversation Texture

Scenes should allow characters to behave like people:

| Texture | Use |
| --- | --- |
| Interruption | Urgency, intimacy, impatience, fear, familiarity. |
| Disagreement | Values, history, pride, mistrust, different priorities. |
| Laughter | Relief, affection, discomfort, denial, surprise. |
| Teasing | Familiarity, testing boundaries, masking concern. |
| Argument | Real stakes, clashing wounds, incompatible truths. |
| Comfort | Loyalty, tenderness, awkward care, emotional growth. |
| Evasion | Fear, shame, secrecy, grief, strategy. |
| Silence | Trust, shock, restraint, refusal, shared understanding. |

## Humor Direction

Humor should come from who characters are:

- What they notice.
- What they misunderstand.
- What they are too proud to admit.
- How they handle fear.
- How they show affection.
- What they find absurd.
- How they deflect pain.

Avoid joke-shaped dialogue that could be moved to any character unchanged.

## Voice Profile Template

Use this for major characters:

| Field | Purpose |
| --- | --- |
| Speech Rhythm | Fast, slow, clipped, wandering, careful, blunt, lyrical, hesitant. |
| Vocabulary | Plain, formal, regional, technical, playful, spiritual, practical. |
| Humor Style | Dry, teasing, awkward, warm, deadpan, dramatic, accidental. |
| Emotional Tell | What changes when they are scared, angry, lying, or moved. |
| Interruption Habit | Do they interrupt, trail off, talk over, wait, redirect, or finish others' thoughts? |
| Conflict Style | Argues directly, jokes, withdraws, bargains, needles, comforts, deflects. |
| Comfort Style | Practical help, quiet presence, jokes, advice, food, touch, stories, promises. |
| Exposition Risk | The kind of information this character might be tempted to explain unnaturally. |
| Recognition Test | Sample lines without speaker labels. |

## Sample Line Test

For each major character, eventually write five unlabeled sample lines:

1. Relaxed.
2. Afraid.
3. Angry.
4. Comforting someone.
5. Hiding something.

If the lines feel interchangeable with another character, the voice is not ready.
