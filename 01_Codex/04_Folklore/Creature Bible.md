## Creature Design Mandate

Creatures should draw inspiration from Filipino folklore while being adapted carefully for gameplay, tone, ecology, cultural respect, and world consistency.

These are original Alamat interpretations, not direct recreations of folklore entries.

Not every creature should be evil. Many creatures should simply exist within the world: feared, respected, misunderstood, avoided, bargained with, protected, hunted, laughed about, or woven into local life.

## Core Philosophy

Creatures are part of the world before they are enemies.

Every major creature should have behavior, habitat, ecological function, local meaning, and a relationship with humans. Combat style comes after identity, ecology, and lore.

## Required Creature Fields

| Field | Purpose |
| --- | --- |
| Legend | What people say about the creature. |
| History | How long the creature has been known, feared, protected, or misunderstood. |
| Behavior | What it does when unthreatened, threatened, hungry, curious, territorial, or social. |
| Habitat | Where it lives and why. |
| Relationship with Humans | Fear, respect, trade, taboo, worship, coexistence, conflict, humor, or misunderstanding. |
| Combat Style | How it behaves mechanically if encountered in battle. |
| Weaknesses | Ecological, behavioral, spiritual, material, emotional, or tactical vulnerabilities. |
| Role in Ecosystem | What it eats, protects, spreads, balances, disrupts, or signals. |
| Visual Design | Silhouette, palette, anatomy, scale, materials, expression, and movement. |
| Sounds | Vocalizations, movement sounds, environmental cues, silence, music relationship. |
| Lore Connections | Links to regions, legends, dungeons, quests, items, magic, characters, or history. |

## Adaptation Rules

1. Respect folklore inspiration without treating it as a monster list.
2. Transform inspiration into original Alamat worldbuilding.
3. Avoid making every folklore-inspired creature hostile.
4. Define ecology before combat.
5. Define local beliefs and contradictions.
6. Let regions disagree about the same creature.
7. Give creatures non-combat value: warnings, rituals, trade, environmental signs, jokes, taboos, or story consequences.
8. Do not use a creature only because it is famous.

## Creature Categories

| Category | Description |
| --- | --- |
| Wild Creatures | Animals or folklore-touched beings that primarily follow ecology. |
| Boundary Creatures | Beings associated with thresholds, paths, bridges, forests, rivers, caves, or ruins. |
| Household or Village Spirits | Presences tied to homes, work, ancestors, offerings, or local etiquette. |
| Dungeon Denizens | Creatures whose ecology is tied to a specific dungeon's history or conditions. |
| Sacred or Taboo Beings | Creatures people avoid, honor, appease, or protect. |
| Predatory Threats | Dangerous beings with understandable needs, hunger, territory, or corruption. |
| Misunderstood Beings | Creatures feared because of distorted legends or human mistakes. |

## Encounter Roles

Combat is only one possible encounter type.

| Encounter Type | Use |
| --- | --- |
| Observation | Player learns behavior before acting. |
| Avoidance | The creature is dangerous but not required to fight. |
| Bargain | The creature responds to offerings, actions, or respect. |
| Puzzle | Behavior, sound, movement, or habitat solves a dungeon problem. |
| Warning | Creature presence signals danger, weather, magic, or taboo. |
| Battle | Combat occurs because of threat, territory, hunger, fear, or story pressure. |
| Boss | The creature embodies a dungeon, legend, wound, or major conflict. |

## Cultural Handling Notes

- Treat Filipino inspiration with respect, authenticity, and thoughtful research.
- Avoid flattening regional beliefs into one universal version.
- Avoid turning sacred or culturally meaningful beings into simple loot sources.
- Preserve mystery where explanation would weaken wonder.
- Local humor is allowed when it comes from people and place, not mockery.

## Creature Profile Template

Use this structure for every major creature:

| Section | Purpose |
| --- | --- |
| Identity | Name, status, canon state, related folklore inspiration. |
| Original Alamat Interpretation | How this version belongs to Alamat. |
| Legend | What people say. |
| History | How the creature fits into memory and world history. |
| Behavior | Non-combat and combat-adjacent behavior. |
| Habitat | Region, biome, dungeon, season, shelter, territory. |
| Relationship with Humans | Fear, respect, coexistence, conflict, taboo. |
| Combat Style | Battle behavior after ecology is defined. |
| Weaknesses | Tactical and world-based vulnerabilities. |
| Role in Ecosystem | Ecological purpose. |
| Visual Design | Silhouette, palette, anatomy, movement, scale. |
| Sounds | Audio identity. |
| Lore Connections | Links to world, story, dungeons, quests, items, magic. |
| Open Questions | Unresolved canon decisions. |
