## Purpose

Skills define character actions learned through growth, practice, relationships, courage, memory, and discovery.

Skills should not exist only to fill class templates. They should express who the character is.

## Skill Design Rules

- Character identity comes before combat role.
- Skills should create memorable battle moments.
- Skills can emerge from story growth, training, humor, fear, loyalty, or personal history.
- Exploration and dungeon skills should reward curiosity.
- Party synergy should reflect relationships, not only mechanical combos.

## Skill Categories

| Category | Purpose |
| --- | --- |
| Personal Skills | Actions tied to a character's personality and history. |
| Relationship Skills | Actions unlocked through trust, conflict resolution, or shared memory. |
| Exploration Skills | Tools for dungeons, travel, secrets, and environmental interaction. |
| Combat Skills | Battle actions, support, defense, tactics, and risk-reward moves. |
| Legend Skills | Rare abilities tied to folklore, ancient places, or story revelations. |
