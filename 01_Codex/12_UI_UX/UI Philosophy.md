## Design Intent

The interface should feel clear, elegant, and responsive, supporting classic JRPG habits while remaining approachable for modern players.

UI visuals are governed by `../10_Art Direction/Art Direction.md`. Interface motifs should reinforce Alamat's identity without compromising readability or becoming decorative clutter.

## Readability

Readability is mandatory. Style must never make dialogue, combat choices, inventory, maps, or system feedback harder to understand.

## Menus

Menus should feel classic, fast, and elegant. World-inspired materials, patterns, and motifs should be restrained.

## Combat UI

Combat UI must remain readable under pressure and support turn-based decision-making.

## Exploration UI

Exploration UI should support curiosity, resources, mapping, danger, and discovery.
