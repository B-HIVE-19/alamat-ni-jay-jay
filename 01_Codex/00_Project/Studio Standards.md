## Purpose

Studio Standards defines how 1910 Creative Studios creates, reviews, names, organizes, and protects work for *Alamat ni Jay Jay*.

This document exists so the project can grow without losing its emotional center, internal consistency, or professional discipline.

## Core Standards

1. Preserve approved canon unless Jeffrey Moralde explicitly revises it.
2. Treat the Alamat Codex as the single source of truth.
3. Record meaningful decisions in the Decision Log.
4. Keep drafts clearly marked as drafts.
5. Separate governance documents, Codex bibles, production deliverables, game implementation, assets, marketing, and archive material.
6. Do not create generic RPG content when specific Alamat content is needed.
7. Respect Filipino inspiration through thoughtful research, original transformation, and cultural care.
8. Protect the tribute heart of the project over spectacle, convenience, or trends.

## Completion Gate

Before any major work is considered complete, ask:

- Would this make 12-year-old Jeff excited to keep playing?
- Would this make someone remember *Alamat ni Jay Jay* twenty years from now?
- Would this honor the emotional reason this project exists?

If any answer is no, keep designing.

## Review Standard

Every reviewed document, asset, prototype, or production deliverable should be checked for:

- Canon consistency.
- Emotional authenticity.
- Adventure value.
- World specificity.
- Production usefulness.
- Clear owner, status, and version.
- Links to relevant AS, AC, PROD, or asset references.
