## Purpose

Milestones tracks major planning phases for *Alamat ni Jay Jay*.

Dates, scope, and staffing remain draft until separately approved.

## Milestone Framework

| Milestone | Purpose | Status |
| --- | --- | --- |
| Project Foundation | Establish studio structure, Codex governance, doctrine, and document families. | In Progress |
| Codex Foundation | Build the major bibles, registers, and production standards. | In Progress |
| Prototype Planning | Define first playable loop, dungeon feel, combat test, and dialogue review workflow. | Draft |
| Technical Prototype | Prove movement, first-person dungeon navigation, turn-based combat, UI, and data loading. | Pending |
| Vertical Slice | Produce a small but emotionally representative playable section. | Pending |
| Production Plan | Define scope, pipeline, tools, schedule, asset needs, QA, and release strategy. | Pending |

## Milestone Approval Test

A milestone is not complete only because files exist. It must make the project clearer, more playable, more emotionally grounded, or more production-ready.
