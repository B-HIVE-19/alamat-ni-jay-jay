# Alamat ni Jay Jay

Project home for the official design, story, world, systems, art direction, references, and prototype notes for *Alamat ni Jay Jay*.

## Codex Purpose

The Alamat Codex is the single source of truth for the game. It should be maintained like a professional studio bible: clear, version controlled, internally consistent, expandable, and respectful of established canon.

## Creative Direction Standard

All documentation should support a cohesive classic JRPG that feels rooted in a believable world beyond the main plot. Design work should consider player experience, production feasibility, thematic consistency, and long-term maintainability.

All creative decisions must follow the approved Alamat Doctrine in `00_Project/Design Pillars.md`.

## Canon Policy

Established lore and approved design decisions must be preserved unless Jeffrey Moralde explicitly changes them. When a new idea conflicts with existing canon, the conflict should be flagged before the Codex is updated.

| Status | Meaning |

| Deprecated | Replaced by a newer decision, preserved for history. |
