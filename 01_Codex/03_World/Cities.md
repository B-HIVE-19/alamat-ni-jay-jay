## City Design Intent

Cities should feel dense with history, class, labor, ritual, commerce, gossip, and contradiction. A city should not be only a large shop menu; it should be a political organism.

## City Profile Template

| Field | Purpose |
| --- | --- |
| City Name | Working or approved name. |
| Founding Reason | Port, fortress, shrine, river crossing, ruin claim, market. |
| Districts | Social and economic geography. |
| Government | Formal authority and informal power. |
| Economy | Goods, guilds, labor, debt, taxes. |
| Architecture | Materials, layout, roofs, bridges, courtyards, defenses. |
| Religious Life | Shrines, rituals, festivals, taboos. |
| Social Tensions | Class, region, trade, law, old history. |
| Key Locations | Inns, markets, docks, archives, gates, hidden places. |
| Story Function | Why the player returns here. |
