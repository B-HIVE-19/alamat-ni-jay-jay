# Dungeon Reviews

Dungeon review workspace.

Use this folder for notes on dungeon purpose, legend, history, layout, atmosphere, puzzle design, secrets, creature ecology, treasure, boss encounters, and exit strategy.

## Review Questions

- Does the dungeon tell its own story?
- Would someone remember this dungeon twenty years from now?
- Is curiosity rewarded?
- Is danger readable and meaningful?
- Does the dungeon feel like a place, not a hallway?
