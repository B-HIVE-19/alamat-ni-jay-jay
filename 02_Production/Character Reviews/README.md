# Character Reviews

Character review workspace.

Use this folder for review notes on character bibles, locks, relationships, timelines, dialogue, concept art, sprites, portraits, and animation personality.

## Review Questions

- Does this character feel like a real person?
- If Jay Jay never existed, what would this person be doing?
- Does the character have a distinct voice, humor, fear, and personal history?
- Would the player miss this person after finishing the game?
- Does combat expression follow personality rather than role?
