# Story Reviews

Story review workspace.

Use this folder for notes on main story, side stories, chapters, themes, emotional moments, foreshadowing, symbolism, and continuity.

## Review Questions

- Does the story move through character choices rather than destiny alone?
- Does the scene create memory rather than only plot movement?
- Is humor balancing darkness naturally?
- Is the tribute heart protected?
- Does the player feel wonder, curiosity, and adventure?
