# 1910 Creative Studios

Working studio structure for *Alamat ni Jay Jay*.

## Studio Mandate

1910 Creative Studios is building an original commercial-quality JRPG. Creative decisions should protect canon, emotional authenticity, cultural specificity, and the identity of Alamat.

The operating mandate lives in `01_Codex/00_Project/Studio Mandate.md`.

## Top-Level Structure

| Folder | Purpose |
| --- | --- |
| `00_Studio` | Studio-level identity, operating notes, governance pointers, and executive references. |
| `01_Codex` | The official game design documentation and single source of truth. |
| `02_Production` | Production planning, product-facing notes, milestone coordination, and deliverable tracking support. |
| `03_Game` | Godot project, prototypes, technical experiments, and runtime implementation. |
| `04_Assets` | Concept art, portraits, sprites, animation, music, sound effects, maps, fonts, icons, UI, videos, references, and production-ready asset sources. |
| `05_Marketing` | Public-facing positioning, store copy drafts, trailers, screenshots, press materials, and launch communications. |
| `99_Canon` | Highest-authority Canon Bible section, canon registries, character art and sheet standards, terminology, timeline, and canon decision log. |
| `99_Archive` | Versions, retired material, historical exports, and preserved decision history. |
