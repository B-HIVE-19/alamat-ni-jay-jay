## Purpose

This document locks the official design system for *Alamat ni Jay Jay* Character Bible pages.

Artists and designers are not creating individual character posters.

They are designing pages from the official Character Bible.

Every character sheet must look like it belongs in the same collector's edition art book.

Consistency is more important than creativity.

The template never changes.

Only the character changes.

### Required Sections

1. Character Name.
2. House Crest.
3. Hero Illustration, largest panel.
4. Character Biography.
5. Expression Sheet.
6. Front / Side / Back Turnaround.
7. Equipment.
8. Signature Ability.
9. Companion, or Character Specialty if no companion.
10. Relationships.
11. Personality.
12. Likes & Dislikes.
13. Fun Facts.
14. Signature Quote.
